Description

Let's start by adding the most important thing to our cinema: the player and the title. From the very beginning, we will make sure that our page with the service for watching movies is SEO-optimized!

Objectives
  1. Let's start by opening the Figma file with the design of the online cinema. You can find all the key elements, their styles, location, and other things there. We hope this will be an exciting quest for you, because working with figma layouts takes a lot of time as a frontend developer. You can use semantic HTML tags for the page markup to improve the SEO optimization;
  2. In the Figma layout on the layers tab, you'll see easter eggs about what kind of IDs, names and tags we expect to see for various elements in the layout;
  3. Do not think about responsive design at this stage — all elements from the layout should be of the same size, color and position as in the Figma file. To test yourself, you can set the size of the window in the browser to the same as on the layout;
  4. Don't forget to add a poster and display controls in your player.
  5. Remember that working with Figma is not just copying styles from the Inspect tab. Your page needs to look exactly the same as it does on the layout, and to do this, sometimes you need to notice something on the layout that is not displayed in the CSS code in Figma.
